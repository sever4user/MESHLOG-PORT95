ЕСЛИ ТЫ ХОСТ (СТРИМЕР ЗВУКА):

🪟 СКАЧАТЬ ДЛЯ WINDOWS (.EXE) 🍏 СКАЧАТЬ ДЛЯ MAC (M1/M2/M3)
⚠️ ИНСТРУКЦИЯ ДЛЯ MAC:
1. Скачай файл и закинь в папку "Загрузки".
2. Открой Терминал на Маке и введи команду:
cd ~/Downloads && chmod +x meshlog-server-mac-arm64 && ./meshlog-server-mac-arm64
3. Нажми Enter. Сервер запустится!

Держи черное окно терминала открытым во время всего джема.
