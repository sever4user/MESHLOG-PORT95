Этот исполняемый файл — локальный сетевой мост для синхронизации
аудио-параметров синтезатора через виртуальную сеть Радмин VPN.

-------------------------------------------------------------------
⚙️ КАК ЗАПУСТИТЬ НА WINDOWS:
-------------------------------------------------------------------
1. Убедись, что ты и твой друг находитесь в одной комнате Радмин VPN.
2. Просто запусти двойным кликом файл `meshlog-server.exe`.
3. Разреши доступ в брандмауэре Windows, если выскочит предупреждение.
4. Не закрывай появившееся черное окно до окончания джема!

-------------------------------------------------------------------
🍏 КАК ЗАПУСТИТЬ НА MAC (M1/M2/M3/M4):
-------------------------------------------------------------------
Система безопасности macOS блокирует сторонний софт без подписи. 
Чтобы выдать права на запуск, сделай три простых шага:

1. Открой стандартное приложение "Терминал" (через Spotlight).
2. Перейди в папку, куда ты скачал файл (например, в Загрузки):
   cd ~/Downloads
3. Разреши запуск файла и запусти его одной строкой:
   chmod +x meshlog-server-mac-arm64 && ./meshlog-server-mac-arm64

Если у тебя старый Мак на процессоре Intel, используй файл с 
именем `meshlog-server-mac-intel`.
